# 🤰 Maternal Care AI

An AI-powered pregnancy risk prediction system that analyzes patient health parameters and predicts risk level using machine learning.

---

## 🚀 Features

- Predicts pregnancy risk (Low / High)
- Uses vital parameters like BP, sugar, temperature, etc.
- Provides confidence score
- Visualizes risk with charts
- Generates downloadable medical report
- Interactive web application (Streamlit)

---

## 🧠 Technologies Used

- Python
- Machine Learning (Random Forest)
- Streamlit (Web App)
- Pandas, NumPy, Scikit-learn

---

## 📊 Model Performance

- Accuracy: ~88%
- F1 Score: ~0.90
- High recall ensures better detection of high-risk cases

---

## 🧪 Input Parameters

- Age  
- Systolic Blood Pressure  
- Diastolic Blood Pressure  
- Blood Sugar  
- Body Temperature  
- Heart Rate  

---

## 💡 Use Cases

- Hospitals and clinics  
- Rural healthcare systems  
- Early risk detection for maternal care
